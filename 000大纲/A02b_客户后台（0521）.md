# 小程序SaaS营销平台 客户管理后台（商户自助端）：功能清单与页面框架

> **设计说明**：本文档为小程序SaaS营销平台「客户管理后台」（商户自助端 Admin Web）的功能清单与页面框架规划。客户管理后台由每个商户客户独立使用，客户登录后可管理自己的小程序、推广位、权益页配置、订阅开关、查看日报数据。
>
> **架构定位**：`总后台（平台运营端） → 客户管理后台（商户自助端） → C端小程序`
>
> 本文档仅覆盖「客户管理后台」。平台运营端的功能清单与页面框架见《A02a_总后台全局说明》。

**模块拆解说明**：

| 模块 | 位置 | 核心内容 | 数据来源 |

| ② 昨日数据 | 顶部左侧 2/3 | 昨日预估佣金(元)（大号字体）+ 昨日订单数，带环比变化，点击可跳转日报详情 | 日报推送模块 |

#### C02 小程序管理

**列表页字段（C02-A）**：小程序名称、AppID、小程序类型、部署状态（从总后台同步）、订阅开关状态（已开启/已关闭）、注册时间、状态（正常/审核中/已停用）、操作（查看详情/编辑）。

**默认排序**：注册时间倒序。

**筛选条件**：小程序名称搜索、状态筛选、订阅开关状态筛选。

**注册/编辑表单（C02-A-a）**：

| 字段 | 类型 | 校验规则 |
|------|------|----------|
| 小程序名称 | 文本（max 30字） | 必填 |
| AppID | 文本 | 必填，全局唯一，格式校验 |
| 小程序类型 | 下拉单选 | 必填（微信小程序/支付宝小程序/抖音小程序） |
| 描述 | 文本域（max 200字） | 选填 |

**交互逻辑**：
- **注册即入驻**：客户通过轻量注册或极速入驻提交小程序信息后，总后台自动新增一条小程序记录。极速入驻未完成的小程序状态为"入驻中"。
- **部署进度同步**：小程序详情页展示从总后台同步的部署进度（授权→名称审核→备案→上线），只读。
- **小程序不可删除**：客户不可删除小程序记录，仅可联系运营停用。

**特殊约束**：AppID全局唯一，重复提交提示错误。

**订阅开关管理（C02-C）**：

订阅开关作为小程序的一个属性，集成在小程序管理模块中。列表页直接展示开关状态并支持行内操作。

**交互逻辑**：
- **默认开启**：小程序注册后订阅开关默认开启，C端用户点击广告位弹出微信订阅消息授权弹窗。
- **关闭**：客户在列表页关闭某个小程序的订阅开关后，C端该小程序用户点击广告位不再弹出订阅弹窗。
- **重新开启**：客户可随时重新开启。
- **批量操作**：支持勾选多个小程序，一键批量开启/关闭订阅开关。

**操作记录（C02-C-a）**：客户查看自己的订阅开关操作日志（操作时间/操作类型/变更前后状态），只读。

**特殊约束**：订阅开关操作记录同步至总后台 B08-B。

#### C03 推广位管理

**列表页字段（C03-A）**：推广位名称、推广位ID、是否为默认推广位（标记）、绑定活动数、创建时间、操作（编辑/设为默认/删除）。

**默认排序**：默认推广位优先 → 创建时间倒序。

**筛选条件**：推广位名称搜索。

**交互逻辑**：
- **系统预置默认推广位**：客户注册成功后，系统自动创建1个默认推广位，名称"默认推广位"，不可删除。
- **新增推广位**：客户点击[新增推广位]，填写名称 → 保存 → 列表新增一行。无数量上限。
- **编辑推广位**：仅可修改名称，推广位ID不可修改。
- **设为默认**：将某个推广位设为默认，原默认推广位自动取消。取链时默认选中默认推广位。
- **删除推广位**：默认推广位不可删除；非默认推广位删除前二次确认。

**特殊约束**：默认推广位不可删除；每个客户至少保留1个推广位（即至少保留默认推广位）。

#### C04 权益页配置（V1.1 极简交互版）

> **设计理念**：用"页面（Tab）"替代"推广位"术语降低商户理解成本，底层仍保留 `slot` 概念。Tab平铺直观展示不同页面，替换坑位活动仅需2次点击（点卡片→点新活动），无需保存确认。通过统一弹窗收口新建与开关控制。

**权益页配置主页（C04-A）**：PC后台左右分栏布局，右侧主内容区为Tab栏 + 状态栏 + 6坑位网格。

**页面结构**：

| 区域 | 内容 |
|------|------|
| Tab栏 | 平铺展示所有"页面"（默认主页 + 自定义页），点击切换；已关闭的Tab置灰显示"(已关闭)"仍可点击编辑；末尾固定 [⚙️ 推广位管理] 入口按钮 |
| 状态栏 | 当前页面启用状态（生效中/已关闭）、推广参数说明（如 `?promoId=t_618`）、[📋 复制推广参数] 按钮 |
| 6坑位网格 | 3×2排列的活动卡片，每卡片展示：活动主图、活动名称、跳转类型标签（🔗可直跳/📋需口令/⚠缺属性） |

**交互逻辑（核心极简）**：
- **Tab切换**：点击Tab切换 `slot_id`，即时刷新状态栏+6坑位网格。
- **替换坑位活动（2次点击）**：点击坑位卡片 → 弹出D01活动选择器 → 点击目标活动 → 即时替换，弹窗自动关闭（无需保存确认按钮）。
- **推广位管理（D02弹窗）**：
  - 勾选/取消Checkbox即时切换页面启用/关闭状态。
  - 默认主页Checkbox锁定为勾选态（disabled），不可关闭。
  - [+新建推广位] → 输入名称（≤20字）→ 创建后自动关闭弹窗，Tab栏新增并自动选中。
- **默认主页兜底**：系统预置"默认主页"（6个活动坑位已填充），始终启用不可关闭。小程序端无参数或参数对应页面已关闭时，回落渲染默认主页。
- **新建页面继承**：新建页面自动拷贝默认主页的6个活动作为初始内容，商户可在此基础上修改。
- **活动取链整合**：活动取链不再独立为单独功能/页面，整合在权益页配置流程中——商户在坑位中选择活动即完成取链。

**弹窗定义**：

| 弹窗ID | 弹窗名称 | 触发条件 | 弹窗内容 | 关闭行为 |
|--------|----------|----------|----------|----------|
| D01 | 活动选择器 | 点击坑位卡片 | 搜索框 + 可用活动列表（单选，已选高亮，标注跳转类型） | 点击活动→即时替换并自动关闭；点击✕→不保存关闭 |
| D02 | 推广位管理 | 点击[⚙️ 推广位管理] | Checkbox列表（页面启用/关闭开关）+ [+新建推广位]按钮 | 点击✕关闭；新建完成后自动关闭 |

#### C05 日报

**列表页字段（C05-A）**：日期、收入(元)、订单数、操作（查看详情）。

**默认排序**：日期倒序。

**筛选条件**：日期范围。

**交互逻辑**：
- **数据由外部统计后台计算**：SaaS系统接收统计后台推送的日报数据，匹配到对应商户账号后分发至客户后台。
- **只读**：日报数据不可编辑。
- **日报详情（C05-A-a）**：点击进入详情页，展示当日汇总（收入/订单数/推广位数量）+ 各推广位明细（推广位名称/收入/订单数）。

**特殊约束**：日报数据按天汇总（列表页）+ 按推广位维度明细（详情页）；当前仅含收入+订单数二维度，后续可扩展；匹配失败时客户后台不展示，由总后台 B09 记录异常。

---

#### C06 账号设置

**企业信息（C06-A）**：查看和编辑企业信息（公司名称、联系人、联系电话等），部分字段（如注册手机号）只读。

**修改密码（C06-B）**：旧密码 + 新密码 + 确认新密码。

**操作日志（C06-C）**：查看自己账号的操作日志（登录记录、关键操作记录），只读。

## 五、总后台 ↔ 客户后台 边界对照表

| 功能领域 | 总后台（平台运营端） | 客户管理后台（商户自助端） | 数据流向 |
|----------|---------------------|--------------------------|----------|
| 客户管理 | 查看所有客户列表、详情、停用/恢复 | 查看/编辑自己的企业信息 | 客户注册 → 总后台 |
| 小程序管理 | 查看所有小程序列表、详情、停用/恢复；查看订阅开关状态及操作日志 | 注册/查看/编辑自己的小程序；开启/关闭订阅开关 | 客户提交/操作 → 总后台同步 |
| 推广位管理 | 查看所有推广位、配置平台参数、停用/启用 | 新增/编辑/删除/设为默认 | 客户操作 → 总后台可查 |
| 活动管理 | 创建/编辑/上下架活动、配置规则 | 查看可参与活动、取链、换活动 | 总后台配置 → 客户可见 |
| 推送管理 | 配置推送模板/任务/人群、查看执行记录 | -（无） | 总后台 → C端用户 |
| 部署管理 | 管控部署全流程、推进节点、备注异常 | 查看自己的部署进度（只读） | 总后台 → 客户可见 |
| 权益配置 | 配置全局默认规则、查阅商户权益 | 配置自己的权益页、换活动、取链 | 总后台规则 → 客户默认生效 |
| 日报 | 管理推送任务、查看推送执行记录 | 查看自己的日报数据 | 统计后台 → SaaS → 客户 |
| 数据看板 | 全平台运营数据 | 自己商户的数据概览 | 总后台全量 / 客户仅自己 |
| 系统设置 | 管理员账号、操作日志 | 修改密码、自己的操作日志 | 各自独立 |
