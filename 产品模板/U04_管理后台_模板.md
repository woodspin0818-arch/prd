## 4. 后台页面布局通用规范

> 填写指引：以下为所有后台页面应遵循的通用布局规范，如有特殊页面需要突破，需在对应页面说明中标注。

| 规范项 | 说明 |
| --- | --- |
| **整体布局** | 左侧固定宽度导航栏 + 右侧自适应内容区 |
| **导航高亮** | 当前所在的一级菜单高亮，展开的二级菜单高亮 |
| **面包屑** | 二级及更深页面顶部显示面包屑导航 |
| **返回按钮** | 表单页/详情页左上角提供"← 返回列表" |
| **保存/取消** | 表单页底部固定或跟在内容末尾，保存和取消按钮并排 |
| **空状态** | 列表无数据时展示空状态插画 + 引导文案 |
| **加载态** | 列表和详情页加载中展示骨架屏或 loading |
| **错误态** | 接口异常展示错误提示 + 重试按钮 |
