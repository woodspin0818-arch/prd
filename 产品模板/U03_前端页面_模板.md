## 3. 常见页面类型参考

### 3.1 Tab页 / 聚合首页

> 适用：底部导航栏的一级页面，聚合多个子模块入口

```
┌──────────────────────────────────────┐
│  [导航栏标题]                         │
├──────────────────────────────────────┤
│                                      │
│  ┌──────────────────────────────┐    │
│  │  [模块A：轮播/展示区域]        │    │
│  └──────────────────────────────┘    │
│                                      │
│  ┌──────────────────────────────┐    │
│  │  [模块B：入口区（图标+文字）]  │    │
│  │  ┌──────┬──────┬──────┐      │    │
│  │  │入口1 │入口2 │入口3 │      │    │
│  │  └──────┴──────┴──────┘      │    │
│  └──────────────────────────────┘    │
│                                      │
│  ┌──────────────────────────────┐    │
│  │  [模块C：列表/内容区域]        │    │
│  └──────────────────────────────┘    │
│                                      │
├──────────────────────────────────────┤
│  [底部导航栏]                         │
└──────────────────────────────────────┘

```

### 3.2 列表页

> 适用：展示数据列表的页面，含筛选/搜索

```
┌──────────────────────────────────────┐
│  [← 返回]           [页面标题]        │

├──────────────────────────────────────┤
│                                      │
│  ┌──────────────────────────────┐    │
│  │  [列表项A：标题/摘要/状态]     │    │
│  │  关键字段1 │ 关键字段2 │ 操作  │    │
│  └──────────────────────────────┘    │
│                                      │
│  ┌──────────────────────────────┐    │
│  │  [列表项B：标题/摘要/状态]     │    │
│  │  关键字段1 │ 关键字段2 │ 操作  │    │
│  └──────────────────────────────┘    │
│                                      │
│         ...（上拉加载更多）...         │
│                                      │
└──────────────────────────────────────┘

```

### 3.3 详情页
