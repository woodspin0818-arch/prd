## 🔴 后台一致性检查清单

> 本模板填写完成后，逐项检查：

- [ ] **功能对照**：后台每个功能ID是否都有对应的 C 端功能ID（引用 A01）？
- [ ] **兜底策略**：配置类功能是否明确了最少保留/默认值等兜底策略？
- [ ] **字段对齐**：后台配置的枚举值、跳转类型等是否与 C 端完全一致？
- [ ] **全链路闭环**：C 端所有需要后台支撑的功能是否都有对应的后台管理入口？
- [ ] **单例配置**：全局唯一配置项（如二维码）是否明确为单例，无列表页？
- [ ] **列表规范**：各类列表页的默认排序、分页、筛选规则是否已定义？
- [ ] **图片规范**：所有涉及图片上传的模块是否已声明尺寸和格式要求？
